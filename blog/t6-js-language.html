Below is an explanation of elements in the javascript language with the use cases for each element.

Parentheses () - Parentheses allow you to group and contain you elements/ operators an to isolate the expression of assignment. It picks up of the sequencing hierachy of something like BODMAS in algebra that would combine elements that need to be processed together

Brackets [] - Brackets contain arrays [ ‘1’ + ‘2’ + ‘3’ ]. With these items in an array we can select and manipulate them by number and do all the other cool things you can do with arrays.

Single quotes '' and Double Quotes “” - Single quotes and double quotes are used to contain strings. This would be more for text that is to be shown on the website rather than interpreted as javascript. You cannot open a string with a single quote and close with a double quote - and vice versa. If you are quoting within a string you could use a single quote to distinguish the ‘script.’
